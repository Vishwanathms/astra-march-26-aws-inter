To install **NGINX** on **Amazon Linux 2023**, follow these steps.

---

## 1. Update the system

```bash
sudo dnf update -y
```

---

## 2. Install NGINX

Amazon Linux 2023 uses **DNF** instead of YUM.

```bash
sudo dnf install nginx -y
```

---

## 3. Start NGINX service

```bash
sudo systemctl start nginx
```

---

## 4. Enable NGINX to start at boot

```bash
sudo systemctl enable nginx
```

---

## 5. Check service status

```bash
sudo systemctl status nginx
```

---

## 6. Check port opened and localhost

```bash
netstat -tlpn | grep 80 

curl localhost
```

---


## 7. Test from browser

Open:

```
http://<EC2-Public-IP>
```

You should see the **NGINX welcome page**.
